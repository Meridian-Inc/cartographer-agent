# Icons

Place your application icons here:

- `icon.png` - Main icon (used for system tray)
- `32x32.png` - 32x32 icon
- `128x128.png` - 128x128 icon
- `128x128@2x.png` - 128x128@2x icon (retina)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

You can generate these from a single high-resolution source image (recommended: 1024x1024 PNG).

For now, placeholder files are created. Replace them with actual icons before building for production.
